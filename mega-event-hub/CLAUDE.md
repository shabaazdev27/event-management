# AI Assistant Instructions for ArenaLink

## Project Overview
ArenaLink Mega-Event Hub is a robust application supporting live event management, crowd density mapping, real-time security alerts, and gamified attendee experiences.

## Technical Guidelines
- **Framework**: Next.js App Router (TypeScript). Respect modern Next.js conventions (Server Components vs Client Components).
- **Styling**: Tailwind CSS. Focus on high-quality, engaging UI/UX with smooth animations and interactive elements.
- **Backend Services**: Real-time updates via Firebase and GCP. AI features powered by the Gemini API.

## Workflow Rules
- Always check the `AGENTS.md` for Next.js specific deprecation context.
- Prioritize concise, modular component design.
- Create engaging UI that avoids generic styling.
